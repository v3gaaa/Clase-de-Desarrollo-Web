Tipos de variables en JavaScript

En JavaScript, las variables pueden almacenar diferentes tipos de datos, como números, cadenas de texto, booleanos, matrices, objetos y más.

Tipos de variables en JavaScript:

  • var: se utiliza para declarar una variable global o local.
  • let: se utiliza para declarar una variable local con un alcance de bloque.
  • const: se utiliza para declarar una constante con un valor que no cambia.

Tipos de Bucles en JS

Bucles son estructuras de control que permiten repetir un bloque de código varias veces.

Tipos de variables en JavaScript:

  • El bucle for: se utiliza para repetir un bloque de código un número específico de veces.
  • El bucle while: se utiliza para repetir un bloque de código mientras se cumpla una condición específica.
  • El bucle do-while: se utiliza para repetir un bloque de código al menos una vez y luego repetirlo mientras se cumpla una condición específica.

Ejemplo de bucle for:

                
                    for (let i = 0; i < 5; i++) {
                        console.log(i);
                    }

Ejemplo de bucle while:

                
                    let i = 0;
                    while (i < 5) {
                        console.log(i);
                        i++;
                    }

Ejemplo de bucle do-while:

                
                    let i = 0;
                    do {
                        console.log(i);
                        i++;
                    } while (i < 5);

Break, continue y etiquetas

La sentencia break termina la ejecución del bucle actual, switch o etiqueta de un bucle.

La sentencia continue termina la iteración actual de un bucle, switch o etiqueta de un bucle y continúa con la siguiente iteración.

Las etiquetas son identificadores seguidos de dos puntos que se pueden colocar delante de las declaraciones de bucle y condicionales para permitir referencias a ellas desde otras partes del programa.

Tipos de Operadores

Los operadores en JavaScript se utilizan para realizar operaciones en variables y valores. Hay varios tipos de operadores en JavaScript, como operadores aritméticos, operadores de asignación, operadores de comparación, operadores lógicos y más. Cada tipo de operador se utiliza para realizar un tipo específico de operación, como sumar dos números, comparar dos valores o asignar un valor a una variable.
